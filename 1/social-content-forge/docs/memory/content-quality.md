# 内容质量标准参考

> 最后更新：2026-03-31

## 目标内容风格

**参照**：Dan Koe写作风格 + 姜胡说内容

## 受众画像

- 大厂边缘人（35岁焦虑、晋升无望、技能错配）
- 小企业主（流量焦虑、经营瓶颈）
- 职场内卷挣扎者（希望找到新方向）

## 爆款内容特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 个人故事切入 | 用「我」的真实经历/脆弱性开头 | 「我右耳完全听不见」 |
| 反常识洞察 | 提供独特视角，挑战主流观念 | 「悲观者永远正确，乐观者总有机会」 |
| 具体案例 | 真实人物/公司/数据支撑 | 贝索斯、冯小刚、张一鸣 |
| 理论与实操结合 | 先讲为什么，再给怎么做 | 自学+48小时定律 |
| 口语化+节奏感 | 短句强调、长短交错 | 「知道，做到，没了」 |
| 递进结构 | 从诊断问题→揭示原因→解决方案 | 七步递进式章节 |

## 差案例特征（避免）

- 公式化开头（「在这个信息爆炸的时代...」）
- 模板填充内容
- 缺乏真实案例
- 没有个人声音
- 短而浅，不深入
