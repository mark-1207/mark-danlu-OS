## 1. PRISM-OS 是什么

PRISM-OS 是一个**智能选题生成工具**，帮你把模糊的创作想法变成具体的文章标题。

### 它能做什么

- 把"我想写点关于AI的东西"这种模糊想法，变成具体的标题候选
- 检查你想到的标题是否已经有人写过
- 帮你发现选题中的逻辑漏洞
- 生成适合公众号和小红书的两种大纲

### 它不适合做什么

- 不能替你写完整文章
- 不能保证标题一定火
- 不能访问互联网（需要你自己配置搜索API）

## 2. 快速开始

### 2.1 环境要求

- Python 3.8+
- curl 命令行工具（Windows 自带）
- `scripts/.env` 文件配置 API 密钥（见第 3 节）

### 2.2 安装步骤

**步骤 1：获取代码**

```bash
# 如果你已经有了代码，跳过这步
cd PRISM-OSv1/skills/prism-os
```

**步骤 2：安装依赖**

```bash
pip install numpy requests
```

**步骤 3：配置 API 密钥**

编辑 `scripts/.env` 文件，填入你的 API key：
```bash
KIMI_API_KEY=你的Kimi密钥
OPENROUTER_API_KEY=你的OpenRouter密钥
ZHIPU_API_KEY=你的智谱密钥
```

**步骤 4：开始使用**

最简单的方式：
```bash
cd scripts
python prism_os.py run "你的话题" --fast --no-ext
```

## 3. 配置教程

### 3.1 配置 API 密钥

**只有一个地方需要改：`scripts/.env`**

```bash
# scripts/.env - PRISM-OS API 密钥集中管理
KIMI_API_KEY=你的Kimi密钥          # 主路径（Moonshot平台）
OPENROUTER_API_KEY=你的OpenRouter密钥  # 付费备用
ZHIPU_API_KEY=你的智谱密钥         # 向量生成用
```

迁移到新机器只需复制这个 `.env` 文件即可。

### 3.2 获取 API Key

#### Kimi（Moonshot）— 主路径

- 注册：https://platform.moonshot.cn
- 国内可用，支持支付宝充值
- 付费模型，按量计费

#### OpenRouter — 付费备用

- 注册：https://openrouter.ai
- 统一接入多种模型，按量计费
- 中国地区可用模型：Qwen、DeepSeek、Gemma、Mistral、Llama（GPT/Gemini/Claude 不可用）

#### 智谱（ZhipuAI）— 向量生成

- 注册：https://open.bigmodel.cn
- 用于 embedding 和相似度计算

### 3.3 搜索 API（现实校验锚）

| 服务 | 说明 |
|------|------|
| Tavily | 优先使用，默认 key 提供基础额度 |
| Firecrawl | Tavily 失败时自动降级 |

## 4. 使用方法

### 4.1 基本流程

```
你说："我想写篇关于AI时代职业发展的文章"
         ↓
PRISM-OS："检测到选题需求，想生成标题吗？（yes/no）"
         ↓
你说："yes"
         ↓
PRISM-OS：[生成标题候选] → [检查是否重复] → [输出结果]
```

### 4.2 触发方式

**方式一：直接命令（最简单）**
```bash
cd scripts
python prism_os.py run "你的话题" --fast --no-ext
```

示例：
```bash
python prism_os.py run "为何一人公司可能是失业群体的翻身之路" --fast --no-ext
```

**方式二：HTTP 监听模式（跨机器使用）**
```bash
python prism_os.py listen
# 服务启动后，从其他机器 POST 触发：
# curl -X POST http://<机器IP>:8080/run -H "Content-Type: application/json" -d '{"text": "你的话题"}'
```

**方式三：Claude Code 自动识别**
当你在 Claude Code 中表达创作想法时，PRISM-OS 会自动介入：
- "我想写篇文章"
- "最近想创作"
- "有什么选题方向吗"
- "帮我想个标题"

### 4.3 执行确认

PRISM-OS 识别到你的意图后，会问你：

```
检测到选题需求。你想用 PRISM-OS 生成标题吗？（yes/no）
```

- 输入 `yes` → 开始生成
- 输入 `no` → 结束对话

## 5. 功能详解

### 5.1 苏格拉底网关（意图澄清）

**作用**：帮你把模糊的想法说清楚

**例子**：
- 你说："最近想写点东西"
- PRISM-OS 会问：
  ```
  请选择创作方向：
  1. 职场发展（加薪、晋升）
  2. AI 与职业（AI 影响、工作方式）
  3. 认知升级（思维模型、学习方法）
  ```

### 5.2 棱镜引擎（标题生成）

**作用**：基于你的想法，生成 12 个候选标题（4 种角度 × 3 个）

**四种角度**：

| 角度 | 说明 | 示例 |
|------|------|------|
| 逆向拆解 | 颠覆常识 | "为什么努力工作反而让你更穷？" |
| 微观切片 | 具体场景 | "当同事靠副业月入 5 万时，他在做什么？" |
| 系统归因 | 指向结构 | "为什么教育系统让聪明人变傻？" |
| 认知脚手架 | 提供方法 | "如何用费曼学习法 3 天掌握新技能？" |

### 5.3 现实校验锚（查重验证）

**作用**：检查标题是否已经有人写过

**标注说明**：

| 标注 | 含义 | 建议 |
|------|------|------|
| 蓝海 | 很少有人写 | 值得尝试 |
| 黄海 | 有一些竞争 | 需差异化 |
| 红海 | 大量类似 | 谨慎 |

### 5.4 V1.5 扩展（素材和大纲）

**Gap Analysis**：检查你的素材是否充足

**双端大纲**：同时生成公众号和小红书两套大纲

### 5.5 V2 扩展（逻辑校验）

**逻辑压力测试**：检测标题是否有逻辑问题

常见问题：
- 循环论证（用结论证明结论）
- 幸存者偏差（只看到成功的）
- 因果倒置（搞错原因结果）
- 滑坡谬误（过度推断）

### 5.6 V3 扩展（智能进化）

**刺客机制**：定期提醒你重新审视旧选题

**知识拓扑**：告诉你哪些话题写多了、哪些没写

**Prompt 变异**：根据你的偏好自动调整

### 5.7 V4 扩展（主动发现）

**认知裂缝捕捉**：自动发现"社会共识与现实不符"的机会

**数字分身**：学习你的品味，帮你初筛标题

## 6. 常见问题

### Q1: 提示"KIMI_API_KEY not set"怎么办？

编辑 `scripts/.env` 文件，填入你的 API 密钥。迁移到新机器时，复制 `.env` 文件即可。

### Q2: 搜索API失败了怎么办？

系统会自动降级：Tavily → Firecrawl。均失败时跳过查重，标题标注为"未校验"。

### Q3: 出现 "curl failed" 错误怎么办？

通常是因为 API rate limit（短时间内调用过多）。系统已内置 0.8s 节流，等待 10-15 分钟后重试。如果持续出现，请检查 `.env` 中的 API key 是否正确。

### Q4: 如何在不同机器之间迁移？

复制 `scripts/.env` 文件到新机器的相同位置即可。所有 API 密钥集中在此文件。

### Q5: OpenRouter 哪些模型可用？

中国地区可用（已验证）：`qwen/qwen-2.5-72b-instruct`、`deepseek/deepseek-chat-v3`、`google/gemma-4-26b-a4b-it`、`mistralai/mistral-small-24b-instruct-2501`、`meta-llama/llama-3.1-8b-instruct`、`qwen/qwen3-8b`。

不可用（403 区域限制）：GPT-4o / Gemini / Claude 系列。

### Q6: HTTP 监听模式怎么用？

```bash
# 启动监听
python prism_os.py listen

# 默认端口 8080，可自定义：
python prism_os.py listen --port 9000

# 从其他机器触发：
curl -X POST http://<IP>:8080/run -H "Content-Type: application/json" -d '{"text": "你的话题"}'
```
