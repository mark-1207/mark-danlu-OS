# D-005: L5 三子项评分（观点锐度 / 思想模型应用 / 事实准确性）

## 状态
已敲定 | 2026-06-15

## 背景
原有 6 维评分（温度/热度/深度/厚度/情绪曲线/知识迁移）不够。需要决定：
- 加什么子项
- 几个子项
- 怎么评分

## 备选方案

### A：加 1 个子项
- 候选：观点锐度 / 事实准确性 / 思想模型应用
- 单点解决最痛问题

### B：加 2 个子项
- 候选组合：观点锐度 + 事实准确性

### C：加 3 个子项（炉选定）
- 观点锐度（解决"既要又要"的废话）
- 思想模型应用（解决"为用而用"的堆术语）
- 事实准确性（解决"为论证而虚构"的 AI 味）

## 选定
**L5 三子项**（观点锐度 / 思想模型应用 / 事实准确性）。

理由：
1. 3 个子项覆盖"言之无物 / 堆术语 / 假数据"3 大 AI 味
2. 每个子项独立评分，可单独报告
3. 配合原 6 维 = 8 维度评分

## 后果

### 正面
- 评分更全面
- 修复建议更具体
- 8 维度 ≥ 7.5 才有意义

### 负面
- LLM 调用次数 +3（草稿后）
- 实现复杂

### 风险
- LLM 评分不稳定
- 缓解：多次采样取中位数

## 评分标准

| 子项 | 检测 |
|------|------|
| **观点锐度** | 强观点密度 / 模糊词密度 / 关键判断数 |
| **思想模型应用** | 模型调用识别 / 贴合度 / 解释度 |
| **事实准确性** | 可追溯 / 虚构警告 / 引用准确 |

**例外**：纯举例/纯客观维度可虚构；核心事实必追溯。
