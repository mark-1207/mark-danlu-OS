# 项目指令

本项目的完整技术文档在 `references/` 目录下：
- `references/CLAUDE_CODE_INSTRUCTIONS.md` — 开发指令和阶段划分（最优先阅读）
- `references/PRD.md` — 产品需求
- `references/TECH_SPEC.md` — 技术规范和接口定义
- `references/ARCHITECTURE.md` — 架构设计
- `references/WORKFLOW_DESIGN.md` — 工作流步骤详细设计
- `references/PROMPT_TEMPLATES.md` — 所有 Prompt 模板

开发时请按 `references/CLAUDE_CODE_INSTRUCTIONS.md` 中的 Phase 顺序执行。
当前阶段开始前，先阅读该阶段涉及的相关文档。
