# 抖音内容策略

## 用户画像
- 年龄：18-40岁，性别均衡
- 场景：碎片时间刷，注意力极短（平均 3 秒决定去留）
- 阅读习惯：视觉 + 听觉同时接收，不爱看文字
- 期望：快速获得信息增量、情绪价值、或娱乐感

## 内容偏好
- 极度精炼：每一句话都要有信息量
- 前 3 秒定生死：必须用反常识/冲突/提问在 3 秒内抓住注意力
- 一个视频只讲一个核心观点：贪多必失
- 节奏感：适合口播朗读，有停顿、有重音、有转折
- 金句收尾：让人想评论或点赞

## 最佳实践
- 标题：引发好奇/冲突/反常识，15-20字
- 前 3 秒：直接抛出核心观点或反常识结论
- 结构：钩子（3秒）→ 核心观点 → 极简案例/类比 → 金句收尾
- 字数：150-300 字（口播 1 分钟以内）
- 节奏：善用 [停顿]、[加重]、[放慢] 标注

## 敏感词/限流词
- 绝对化：绝对、一定、保证
- 医疗："治愈"、"瘦身"
- 金融："稳赚"、"翻倍"
- 虚假宣传：免费、立即等

## 互动引导
- 结尾抛出问题："你认同吗？"
- "点赞 + 关注，下次给你讲..."
- 争议性话题引发评论

## 发布时间建议
- 早：6:00-8:00（通勤）
- 中：12:00-13:30（午休）
- 晚：18:00-20:00（下班）
- 夜：21:00-23:00（睡前）
