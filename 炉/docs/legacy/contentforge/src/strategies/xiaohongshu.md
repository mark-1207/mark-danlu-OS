# 小红书内容策略

## 用户画像
- 年龄：18-35岁女性为主
- 场景：午休、排队、睡前刷
- 阅读习惯：快速浏览，注意力有限，依赖视觉和标题
- 期望：获得可操作的干货、真实的使用体验、情感共鸣

## 内容偏好
- 真实感：喜欢"真实体验"而非"官方背书"
- 可执行：每个 tip 都要具体到"今天就能做"
- 视觉友好：善用 emoji、分割线、表格
- 情感共鸣：喜欢"我"的视角，分享真实经历

## 最佳实践
- 标题：必须包含数字/结果/人群标签中的至少两个
- 人设开头：一句话建立可信度（身份 + 成就/经历）
- 干货点：3-5 个，每个点配 1-2 个 emoji
- 段落：每段不超过 3 行，文字精简
- 结尾：收藏/关注引导金句 + 相关 hashtag（5-8个）

## 敏感词/限流词
- 夸大宣传："最好用"、"绝对有效"
- 医疗效果："治疗"、"减肥"
- 金融："稳赚"、"收益率"
- 平台限流词：免费、领取等营销词

## 互动引导
- 结尾："收藏起来慢慢看"、"关注不迷路"
- 评论区互动：问"你们有没有类似经历"
- 善用投票、问答卡片

## 发布时间建议
- 早通勤：7:30-9:00
- 午休：12:00-13:30
- 晚间：20:00-22:00
