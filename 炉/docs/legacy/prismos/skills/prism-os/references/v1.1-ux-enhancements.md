# 决策点 UX 增强（v1.1.2, 2026-06-12）

> 5 个决策点的 ⭐ 推荐 + 💡 差异说明（1 次 LLM 给方向）+ 截断解除

## 总览

| 决策点 | 加什么 | 实现 |
|--------|--------|------|
| **1.5 DirectionSelect** | ⭐ + 💡 差异说明（1 次 LLM） | 1 次 LLM 生成 N 句差异说明（~30 字/方向）+ 第一个标 ⭐ |
| **1 Prism 标题** | ⭐ + 解除 50 字截断 | 规则（sort_score 最高）+ 完整显示 |
| **1 deep 模式** | archetype 中文标签 + 描述 | 规则（基于 `title_deep.VALID_BASED_ON`）|
| **3 Gap** | ⭐ 推荐 | 规则（readiness 阈值）|
| **0.5/2** | 不改 | 自解释（问题本身清楚）/ 选项含义明确 |

## 决策点 1.5 方向选择

**Before（v1.0）**：
```
━━━ 切入方向选择 ━━━
  1. AI 让某些职业消失（失业焦虑）
  2. AI 时代新职业机会
  3. 职业选择方法论转变
━━━━━━━━━━━━━━━━━━━━
```

**After（v1.4）**：
```
━━━ 切入方向选择 ━━━
  1. AI 让某些职业消失（失业焦虑） ⭐
     💡 聚焦被替代的恐惧视角，与其他方向的乐观基调形成对比
  2. AI 时代新职业机会
     💡 侧重红利与机会捕捉，强调如何抓住
  3. 职业选择方法论转变
     💡 聚焦方法论层面，强调旧方法失效的判断
━━━━━━━━━━━━━━━━━━━━
请选择切入方向（输入编号，或 skip 用原命题继续）:
```

**实现**：
- `socratic_gateway.generate_direction_differences(thesis, directions)` — 1 次 LLM，~30 字/方向
- `phases/direction_select._format_prompt(directions, differences)` — 渲染
- **风险**：1 次额外 LLM（但用户决策质量提升明显，**值得**）

## 决策点 1 Prism 标题

**Before（v1.0）**：
```
  5. ✓ 35 岁程序员，焦虑的不是 AI
     HKR=0.50 | reversal | opinion_assertion
     字数: 22 | 最高相似度: 0.00
     理由: 挑战"35岁必焦虑"的常识假设...
```
（理由截断 50 字）

**After（v1.4）**：
```
  1. ✓ 35 岁程序员，焦虑的不是 AI ⭐ 推荐
     HKR=0.50 | 张力=4 | reversal | opinion_assertion
     字数: 22 | 最高相似度: 0.00
     理由: 挑战"35岁必焦虑"的常识假设（展开论证：程序员焦虑的根源是身份认同危机，
     而不是 AI 替代）。对比其他方向，这个角度从"焦虑错位"切入...
```

**实现**：
- `phases/prism._format_title_prompt` 第一个候选加 ` ⭐ 推荐`
- `rationale[:50]` 截断移除
- **风险**：零（仅展示增强）

## 决策点 3 Gap

**Before（v1.0）**：
```
请选择: [1] 补充素材 [2] 调整大纲 [3] 直接生成 [q] 退出
```

**After（v1.4）**：
```
请选择: [1] 补充素材 [2] 调整大纲 [3] 直接生成 [q] 退出（⭐ = [2]）
```
（readiness < 70% → ⭐ = [2]；≥ 70% → ⭐ = [3]）

**实现**：
- `phases/gap._format_gap_prompt` — 规则判定 readiness 阈值
- **风险**：零（仅显示增强）

## UX 原则

- **真价值**：用户能据此做决策（⭐ 一眼看出推荐，💡 提供决策依据）
- **不加价值** ≠ 噪声：例如 0.5 追问本身已经自解释，加元信息反而冗余
- **规则优先**：能用规则生成的（⭐ 位置、archetype 描述、readiness 阈值）就用规则，避免新 LLM 调用
- **必要时才 LLM**：方向差异说明用 LLM（1 次调用换真实决策依据，不能用规则复制）
