## M3b 系列模式（详解）

**触发**：`python prism_os.py run "<命题>" --series`

**流程**：
1. 主 run 跑完（含 narrate 落盘文章）
2. 锚点 = 刚选中的 T1 标题
3. `series.analyze_suitability(thesis, persona)` — 5 维 LLM + 硬规则
4. 不适合 → 一行原因 + 结束
5. 适合 → 生成 3 个后续标题（带 positioning: deepen/expand/contrarian）
6. 用户选 1-3 / s（全存）/ m（重生成）/ q（跳过）
7. 选中或全存 → 写 `topic_log` 含 `series.{series_id, planned_titles: [...]}`

**硬规则**（覆盖 LLM）：
- `time_sensitivity >= 4` → 强制 unsuitable（时效新闻不适合系列）
- `depth_potential <= 2` → 强制 unsuitable（命题太浅）

**冲突豁免**：`conflict_detect` 同 `series_id` 的历史项不计入命题撞车

**续写**：
```bash
python prism_os.py run "<T2 命题>"  # 跑 T2
python prism_os.py mark_written --series-id <sid> --slot 2  # 标记 T2 已写
python prism_os.py run "<T3 命题>"  # 跑 T3
python prism_os.py mark_written --series-id <sid> --slot 3  # 标记 T3 已写
```

## M4 A1 张力评分

**集成点**：`prism_engine.generate_dimension_titles` 和 `title_deep.DEEP_TITLE_GEN_PROMPT` 的 prompt 末尾加 A1 段：

```
# A1 认知张力自评（每个标题打 1-5 分）
- 5: 强烈对立，挑战主流共识
- 4: 明确反方/新视角
- 3: 一定观点倾向
- 2: 接近通用建议
- 1: 完全信息陈述
```

**解析**：每个 candidate dict 加 `tension_score: int (1-5)` 字段，缺失默认 3

**排序权重**：
```python
# v1.1 拍脑袋：0.6/0.4（v2 接入真实数据后调）
sort_score = tension * 0.6 + consistency * 5 * 0.4
# A1 主导：5×0.6=3.0 vs A3×5×0.4=2.0
```

## M5 A2 维度覆盖

**集成点**：`phases/prism.py` `_format_title_prompt` 顶部加覆盖报告：

```
[维度覆盖] ✓ reversal(2) ✗ contrarian(0)
缺失: contrarian
```

**`r` 命令**：调 `coverage.regenerate_missing_dimensions(missing_dims, thesis)`，复用 `prism_engine.generate_dimension_titles`

**广度/深度模式**：
- 广度 (4 维): reversal / benefit_anchor / micro_scene / contrarian
- 深度 (5 原型): opinion_assertion / identity_label / scene_suspense / data_counter_ask / story_hook

## M6 A3 风格连贯性

**集成点**：
- `storage.append_log` 写入时存 `_title_embedding`（避免每次重算）
- `title_scoring.compute_style_consistency(title, history_titles, top_n=20)` — 与最近 20 条历史标题的余弦相似度均值

**为什么 20**：
- 太小（5）→ 太局部，受最近 1-2 篇波动
- 太大（100）→ 太全局，漂移检测不到
- 20 = 中期窗口（约 1-2 个月发文频率）

**返回**：(score ∈ [0,1], explain)，无历史 → 0.5 中性分

## M9 C1 风格指纹

**集成点**：
- `style_fingerprint.StyleFingerprint.from_persona(persona)` → 提取 4 字段（style_mentors / style_keywords / tone_keywords / avoid_keywords）
- `style_fingerprint.check_style_match(text, fp)` → 规则化 L5 检查
- `style_fingerprint.inject_style_guidance(prompt, persona)` → 注入到 narrate prompt

**修订**：子串匹配最小长度 ≥ 2 字符，避免单字误中

**L5 集成**（content_generator.quality_check）：
```python
if persona:
    from style_fingerprint import StyleFingerprint, check_l5_style
    fp = StyleFingerprint.from_persona(persona)
    style_check = check_l5_style(article_text, fp)

```

## C2 密度控制

**定量**（0 LLM）：
- `count_entities(text)` — 数字 / 年份 / 百分比 / 概念词
- `quantitative_density(text)` — entities / len(text) × 100

**定性**（1 次 LLM）：
- `measure_density(text, persona)` — 2 项核心检查（topic_logic_chain + value_increment）
- `DENSITY_EVAL_PROMPT` 问 2 个 yes/no + suggestions

**权重**：定量 0.6 + 定性 0.4（**反向**：定量主导，LLM 补充）

**L6 集成**（content_generator.quality_check）：
```python
if persona:
    from density import measure_density, check_l6_density
    report = measure_density(article_text, persona)
    density_check = check_l6_density(report)
    # suggestions 转为 issues（severity=warning）
```
