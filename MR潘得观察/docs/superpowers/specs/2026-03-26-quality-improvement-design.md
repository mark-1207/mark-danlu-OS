# 质检准确性优化设计

**日期**: 2026-03-26
**目标**: 解决质检模块的 A3/A1/A4/A2 问题
**状态**: 已批准，实施中

---

## 问题分析

| 问题 | 根源 |
|-----|------|
| A3 维度定义模糊 | 判定标准只有文字描述，无标杆示例 |
| A1 误判严重 | AI无法精准执行抽象标准 |
| A4 优化建议不实用 | 输出冗长，缺乏可操作性 |
| A2 评分不稳定 | 提示词复杂，AI注意力分散 |

---

## 解决方案

### 核心思路

1. **每个维度添加正反例标杆** - 让AI有参照物，减少理解偏差
2. **评分必须引用原文** - 强制说理，减少主观臆断
3. **优化建议改为对照表** - "原文片段 → 修改后片段"，可直接操作
4. **精简输出结构** - 去掉低价值内容（确认信息、终稿等）

## 公众号质检提示词改动

### 维度判定标准（添加标杆示例）

#### 1. 标题/摘要传播性（25分）

**标杆示例**：

```
✅ 达标（20-25分）：
  示例1：原文"我做了3年公众号，终于敢说真话了"
  判定：戳中从业者不敢说真话的痛苦，有态度有传播性
  示例2：原文"别再被公众号培训割韭菜了"
  判定：反向操作+利益提醒，有社交货币

⚠️ 待优化（10-19分）：
  示例1：原文"公众号运营的5个技巧"
  判定：有干货但无情绪调动，缺乏传播钩子
  示例2：原文"如何写好标题"
  判定：太宽泛，未戳中具体痛点

❌ 不达标（0-9分）：
  示例1：原文"分享我的运营经验"
  判定：完全没有钩子，太个人化
  示例2：原文"关于公众号的一些思考"
  判定：无价值点，无传播性
```

**评分必填字段**：
- `score`: 得分
- `evidence`: 必须引用原文片段
- `reason`: 判定理由

---

#### 2. 人群精准度（15分）

**标杆示例**：

```
✅ 达标（12-15分）：
  示例1：原文"做公众号涨不到1w粉的可以看下去，其他人划走"
  判定：明确筛选目标用户，前300字完成人群过滤
  示例2：原文"如果你也在职场中遇到过XX问题"
  判定：精准定位目标人群画像

⚠️ 待优化（6-11分）：
  示例1：原文"今天讲讲职场沟通"
  判定：提了赛道但没筛用户，泛化了

❌ 不达标（0-5分）：
  示例1：原文"给大家分享一些经验"
  判定：完全没有目标人群概念
```

---

#### 3. 社交货币属性（25分）

**标杆示例**：

```
✅ 达标（20-25分）：
  示例1：原文"转发给你身边想做自媒体的朋友，少走3年弯路"
  判定：明确的转发触发点，转发者有面子
  示例2：原文"踩过这5个坑的公众号编辑，默默转了"
  判定：可炫耀的干货+可转给特定人群

⚠️ 待优化（10-19分）：
  示例1：原文"这些都是我的个人经验总结"
  判定：有价值但没有明确转发点

❌ 不达标（0-9分）：
  示例1：原文"今天天气不错，写篇文章"
  判定：完全是碎碎念，无可传播内容
```

---

#### 4. 内容密度（20分）

**标杆示例**：

```
✅ 达标（16-20分）：
  示例1：每300字有1个案例/数据/方法点，无废话
  判定：信息密度高，排版符合手机阅读

⚠️ 待优化（8-15分）：
  示例1：原文"我觉得...我认为...其实呢..."
  判定：大量个人感慨，稀释了信息密度

❌ 不达标（0-7分）：
  示例1：通篇大道理，无具体案例
  判定：空泛，无实际价值
```

---

#### 5. 留存引导设计（15分）

**标杆示例**：

```
✅ 达标（12-15分）：
  示例1：原文"如果觉得有用，点个星标"
  判定：明确星标引导，自然不生硬
  示例2：原文"往期精选文章：[链接]"
  判定：有往期推荐，引导沉淀

❌ 不达标（0-5分）：
  示例1：通篇无任何引导
  判定：用户看完就走，无法沉淀
```
